# アプリ名：dev-nikki

開発の記録を行うアプリです。
「開発記録を残して振り返られるようにする」というコンセプトから<br>
「dev-nikki」というサービス名にしました。

# 概要

2点あります。

1. 新たに学んだことやつまづいた点を整理できること
   プロジェクト毎に新たに学んだこと、つまづいた箇所をまとめておくことにより
   整理されて次に同じような状況になった際に問題を速く解決することができます。

2. 開発の振り返りができること
   例
   どの実装にどのくらいの時間を要したかの記録などの記録をつけたりすることで、
   実装前の見積もりとの差異がわかります。
